# PokemonLive

## Description
A Pokémon web application built with React and Vite that displays Pokémon data using an API.

## Features
- Fetch Pokémon data from API
- Display Pokémon information
- Responsive design

## Tech Stack
- React
- JavaScript
- HTML
- CSS
- Vite
- REST API

## Installation

```bash
npm install
npm run dev
```
